
# OP_NET Charity Sender

A clean, minimal, OP_NET‑styled Bitcoin charity sender.  
Select a charity → auto-fill BTC address → set amount → send BTC (BIP21).

## 1. Features
- Dropdown list of multiple charity options  
- Auto‑filling BTC address (editable)  
- BTC address validation  
- Send BTC using BIP21 link  
- Progress bar animation  
- Neon OP_NET-inspired visual theme  
- Fully static → runs on GitHub Pages and any VPS  

## 2. How to Use it
1. Choose an organization  
2. BTC address is auto-filled (you can change it)  
3. Enter amount  
4. Validate the address  
5. Click **Send BTC**  
6. Your wallet opens automaticall
